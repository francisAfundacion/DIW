PSEUDOCLASE

DAN ESTILOS A LOS DISTINTOS ESTADOS DE LOS ELEMENTOS

Se represeta con selector : nombre_pseudoclase.Ejemplo:
div:hover

Link a todas als pseudoclases

Usos:

  • Para cambiar los estados de los enlaces
    1. No visitado => :link
    2. Visitado => :visited
    3. Activado => :activo (click sin soltar)
    4. pasar el puntero por encima => :hover
    5. TIENE QUE CUMPLIR ESTE ORDEN AL DAR LOS VALORES A ESTAS PSEUDOCLASES
  • Para cambiar estados de los estados de los items de las listas :
    1. first:child
    2. last-child
    3. only-child
    4. nth-child(3)
  • Para cambiar estados de los formularios
    1. :focus => cambia estilos formularios, cuando tiene el foco la entrada concreta
    2. :required =>
    3. :disabled
  • Para cambiar estados de las filas y columnas de las tablas
