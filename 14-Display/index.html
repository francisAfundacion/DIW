VALORES DISPLAY:

  • valor por defecto acorde a la naturaleza del elemento
  • visibility => hace lo mismo que none , pero deja hueco del elemetno
  • none : no se muestra,no deja el hueco del elemento que se oculta
  • Block:elemento se muestre como si fuera un elemento en bloque
  • Inline:Se muestre como si el elemento como fuera uno de tipo line
  • Inline-block:se muestra como si fuera medio bloque, medio line
