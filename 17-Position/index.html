POSITION

Permite posicionar elementos html en cualquier lugar de la pantalla.

Valores position

  • static => es el valor por defecto,Los elementos siguen el flujo de colocación que tienen que seguir por su naturaleza de tipo de elemento.
  • absolute => se puede mover, pero no desde su posición del flujo normal.

    Hay que comprobar si tiene padre y si este padre está posicionado(que tiene un valor de position cualquier, distinto de static)

    Si tiene padre, se mueve dentro de padre

    Si no tiene padre o no está posicionado, se mueve en relación a la pantalla

    Tambien se mueve con las propiedades top,bottom,left y right.

    en absolute no deja hueco AL MOVER EL ELEMENTO

  • relative => se puede mover.Se mueve desde su posición en el flujo normal.

    Como se mueve? con las propiedades top,bottom,right,bottom. Cuándo movemos el elemento queda el hueco o no? => SI LO DEJA

  • Ffixed,se puede mover con top, bottom,left,right.Se mueve en relación a la pantalla y de donde lo movamos se queda fijo.

    Aunque hagamos scrol, el elemento se queda en el lugar especificado

    Deja hueco al mover el elemento? NO DEJA HUECO

  • Se comporta como relative y fixed.

    Inicialmente ocupa su pos normal.

    relative => mientras no lleguemos con scrol a la posición determinada especificado con top,bottom,right.

    fixed => cuando lleguemos a esos valores se fijará en los valores que le digamos
